无协议报错

<?php 
$url = $_GET['url'];
$data = parse_url($url); 
var_dump($data);
?>

该函数在URL没有协议的时候，加上端口会报错

不加端口会正常进行解析

其他情况下可以正常解析

当端口有字母时，依然会正常解析
